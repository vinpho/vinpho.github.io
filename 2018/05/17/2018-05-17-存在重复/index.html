存在重复

2018.05.17 vinpho 算法  热度 ℃

题目

给定一个整数数组，判断是否存在重复元素。

如果任何值在数组中出现至少两次，函数返回 true。如果数组中每个元素都不相同，则返回 false。

示例1

输入: [1,2,3,1]
输出: true

示例2

输入: [1,2,3,4]
输出: false

示例3

输入: [1,1,1,3,3,4,3,2,4,2]
输出: true

第一种做法：

利用循环遍历数组，使每个元素与其他各个元素分别比较。
此方法消耗的时间较长。

利用java的数组排序函数，先将数组排好顺序，然后比较相邻两个元素是否重复即可。
